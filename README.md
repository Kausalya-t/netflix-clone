**Netflix Clone**
A responsive front-end clone of the Netflix homepage built using HTML5 and CSS3. This project replicates the visual design and layout of Netflix, showcasing modern web development techniques, responsive design, and component-based styling.

**Technologies Used**
- HTML5 – Used for structuring the content and layout of the webpage.
- CSS3 – Used for styling, layout design, and responsiveness.
- Flexbox and Grid – Applied for building flexible and responsive layouts.
- Media Queries – Ensured the design adapts to different screen sizes.
- Google Fonts – Integrated custom fonts like Poppins for better typography.
- SVG Icons – Used for scalable and crisp icons in the FAQ section.

**Features**
- Netflix-inspired dark theme
- Mobile-first responsive design
- Hero section with background image and CTA
- Movie thumbnails and video overlays
- FAQ section with interactive hover effects
- Navigation bar with logo and language/sign-in buttons
- Kids profile section with custom visuals
- Footer with multi-column layout and links
